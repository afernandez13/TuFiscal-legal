Política de Privacidad de TuFiscal

Última actualización: 8 de junio de 2026

TuFiscal respeta la privacidad de sus usuarios. Esta política explica cómo se recopila, utiliza y protege la información cuando utilizas nuestra aplicación.

1. Información que recopilamos

TuFiscal puede permitir al usuario subir o analizar facturas, documentos o información relacionada con gastos. Esta información se utiliza únicamente para ofrecer las funciones principales de la aplicación.

2. Uso de la información

La información se utiliza para analizar facturas, clasificar gastos, mostrar resultados dentro de la aplicación y mejorar la experiencia del usuario.

3. Compartición de datos

No vendemos datos personales de los usuarios. La información solo podrá compartirse con proveedores técnicos necesarios para el funcionamiento de la aplicación, como servicios de almacenamiento, autenticación, análisis de documentos o inteligencia artificial.

4. Seguridad

Aplicamos medidas razonables para proteger la información del usuario frente a accesos no autorizados, pérdida o uso indebido.

5. Eliminación de datos

El usuario puede solicitar la eliminación de su cuenta y de los datos asociados enviando un correo electrónico a: soporte@tufiscal.es

6. Contacto

Si tienes dudas sobre esta política de privacidad, puedes contactar con nosotros en: soporte@tufiscal.es
